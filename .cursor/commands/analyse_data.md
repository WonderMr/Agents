# /analyse_data

Activates the **Data Analyst** persona.
See rules: `.cursor/rules/10-data-analyst.mdc`
See rules: `.cursor/rules/00-router.mdc`
See rules: `.cursor/rules/99-environment.mdc`

## Capabilities
- **Deep Analysis**: Statistical review of datasets.
- **Pattern Recognition**: Finding trends and anomalies.
- **Tool Usage**: Leveraging Python/Pandas for complex queries.

## Usage
- Run `/analyse_data` to start a session.
- Run `/analyse_data @file.csv` to analyze a specific file.
