# /research

**Syntax**: `/research [query]`
**Alias**: `/deep`

## Description
Initiates a deep dive research session on the provided topic.

## Behavior
1.  **Analyze**: Deconstructs the query into sub-topics.
2.  **Search**: Performs recursive searches (Web + Academic).
3.  **Filter**: Applies trust weighting (Academic=100%, Web=50%).
4.  **Synthesize**: Generates a dense, summary report (80/20 principle).

## Example
`/research impact of intermittent fasting on longevity`
